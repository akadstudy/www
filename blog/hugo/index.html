Build a Website with Hugo

Here you’ll learn how to build a Website with Hugo, one of the world’s fastest framework for building websites.

Prerequisite

You’ll need some basic knowledge in bash and using terminal. The examples will work on a Mac Book Pro with this tools:

  • Hombrew
  • iTerm
  • OhMyZSH

Install Hugo

Install brew

  1. Paste this into your Terminal:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
  1. First update:
brew update
  1. Install Hugo:
brew install hugo

Create Site

Hugo has commands that allows to quickly scaffold a Hugo manged website.

  1. Navigate to a convenient location on your filesystem and create a new Hugo site mywebsite.
hugo new site mywebsite
  1. Change directory to mywebsite

Add content

  1. Add a post to your mywebsite. For this use the hugo new command to add a post.
hugo new post/my-first-post.md

Note: Make sure you are inside the mywebsitedirectory.

The content inside the my-first-post.md file looks similar:

---
date = "2020-02-14T16:11:58+05:30" 
draft = true 
title = "some title" 
---

Serve content

To start the server on port 1313 you run:

hugo server

You can view the blog at http://localhost:1313/

Note: When you go to the link, you will see nothing.

To render drafts, re-run the server with the flag -D:

hugo server -D

If you got to http://localhost:1313/, you still will not see anything as you not specified a theme that Hugo should use.

Add theme

Themes provide the layout and templates that will be used by Hugo to render your website. There are a lot of Open-Source themes available. Simply pick whichever theme suits your project.

  1. Themes should be added in the themes directory inside the repository root.
cd themes
  1. Here, you can clone one or more themes inside the themes directory.
git clone <https://github.com/REPO/THEMENAME.git>
  1. Leave the themes folder.
cd ..
  1. Open config.toml in your favorite text editor, and replace the contet with the following:
baseURL= "/"
...
theme = "<THEMENAME>"
...

Note: The Name must be same as cloned into the theme folder.

  1. Start the server again:
hugo server

Congratulation your site is up and running!

Summary

If you are comfortable with the commandline you learned here a fast way to install Hugo. Then you created your site with the first post and learned to server the content at http://localhost:1313. Last you attached any of your favorite theme to your hugo page. You can edit the config.toml file as you need and go on from here and tweak your theme as you like.
